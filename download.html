System Requirements

Minimum Requirements

  • 64-bit processor
  • 4GB RAM
  • 20GB storage
  • USB port for installation

Recommended

  • Multi-core processor
  • 8GB RAM or more
  • 50GB SSD storage

Supported Hardware

  • UEFI and Legacy BIOS
  • Common WiFi adapters
  • Standard GPU drivers
